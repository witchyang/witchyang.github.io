ubuntu下卸载python2和升级python3.5

发表于 2017-05-28 | 分类于 python

ubuntu下卸载 python2 只需要一条语句就可以实现：

sudo apt-get remove python

ubuntu下安装python3：
sudo apt-get install python3

但这样只安装了python3.4，要想使用python3.5，则必须升级python3.4：

启动时

输入python时启动的是python2.7

输入python3时启动的是python3.4

输入python3.5时启动的才是python3.5
